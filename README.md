# 🏦 Loan Approval System (End-to-End ML Project)

An end-to-end **Machine Learning pipeline** that predicts loan approval status and estimates loan-related outcomes using classification and regression models.

---

## 🚀 Project Overview

This project builds a complete ML workflow for **loan approval prediction**, including:

* Data ingestion & preprocessing
* Feature engineering
* Model training (Classification + Regression)
* Model evaluation
* Deployment-ready pipeline using Streamlit

---

## 📂 Project Structure

```
loan_approval_system/
│── app/                 # Streamlit frontend
│── config/              # Configuration files
│── data/                # Raw & processed data (ignored in repo)
│── models/              # Trained ML models
│── notebooks/           # EDA & experimentation
│── reports/             # Visualizations & evaluation plots
│── src/                 # Core ML pipeline code
│── tests/               # Unit tests
│── requirements.txt     # Dependencies
│── README.md            # Project documentation
```
